# Nano Banana Logo Prompt for ConsultFlow

## Prompt to Copy and Paste into Nano Banana

```
Modern minimalist logo for "ConsultFlow" - an AI-powered consulting workflow automation tool. Clean vector-style mark on dark background. Abstract flow symbol: interlocking arrows or connecting nodes forming a subtle "CF" or workflow pattern, indicating multi-agent collaboration. Use a gradient from electric indigo to cyan for a tech/AI feel. Include the brand name "ConsultFlow" in clean sans-serif font, bold weight. Style: flat vector, SVG style, white background option available, minimalist, professional. --no realistic photo, shading, complex details
```

## Alternative Prompt (Simpler)

```
Logo for ConsultFlow: simple abstract icon representing agent collaboration - two circles connected by flowing lines forming a subtle loop. Brand name "ConsultFlow" in modern geometric sans-serif. Electric indigo and cyan gradient. Dark background. Flat vector style, minimalist, tech startup aesthetic.
```

## Usage Instructions

1. Go to **nanobanana.org** or open **Gemini** (gemini.google.com)
2. Click the image generation icon (banana icon)
3. Paste one of the prompts above
4. For best results, add your own refinements after generation:
   - "make the text bolder"
   - "add a subtle glow effect"
   - "create a variation with purple and green"

## Tips

- Always put text you want in quotes: `"ConsultFlow"`
- Use `--no` to exclude unwanted elements
- Add "isolated on dark background" for transparent-ready output
- For logo files, ask for "SVG" or "vector" format
